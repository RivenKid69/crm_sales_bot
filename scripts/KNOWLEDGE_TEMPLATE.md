# Шаблон для расширения базы знаний Wipon

## Как добавить новую секцию

Откройте файл `src/knowledge/data.py` и добавьте новую `KnowledgeSection`:

```python
KnowledgeSection(
    category="<категория>",      # products, features, integrations, pricing, etc.
    topic="<уникальный_id>",     # краткое имя темы (латиницей)
    keywords=["<слово1>", "<слово2>", ...],  # ключевые слова для поиска
    priority=5,                  # 1-10, выше = важнее
    facts="""
<Факты о теме - будут переданы в LLM>
    """
),
```

---

## Категории

| Категория | Описание |
|-----------|----------|
| `products` | Продукты Wipon (Kassa, Desktop, Pro, etc.) |
| `features` | Функции (склад, отчёты, кадры) |
| `integrations` | Интеграции (Kaspi, 1C, банки) |
| `pricing` | Тарифы и цены |
| `audience` | Целевая аудитория |
| `benefits` | Преимущества |
| `competitors` | Сравнение с конкурентами |
| `support` | Поддержка и обучение |
| `contacts` | Контакты |

---

## Правила написания фактов

### ✅ Хорошо:
- Конкретные цифры: "12,000₸/год", "50,000+ клиентов"
- Списки с буллетами: "• Функция 1\n• Функция 2"
- Таблицы для сравнений
- Краткие предложения

### ❌ Плохо:
- Размытые формулировки: "хорошая система", "удобный интерфейс"
- Слишком длинные тексты (LLM обрежет)
- Информация без источника
- Устаревшие данные

---

## Правила для keywords

1. **Включайте все формы слова**: `["касса", "кассу", "кассы", "кассой"]`
2. **Добавляйте синонимы**: `["цена", "стоимость", "прайс", "тариф"]`
3. **Учитывайте опечатки**: `["каспи", "kaspi", "касби"]`
4. **Добавляйте разговорные варианты**: `["сколько стоит", "почём"]`

## Пример добавления новой секции

### Задача: Добавить информацию о системе лояльности

**Шаг 1:** Собираем информацию с сайта wipon.kz

**Шаг 2:** Создаём секцию:

```python
KnowledgeSection(
    category="features",
    topic="loyalty_program",
    keywords=[
        "лояльность", "бонус", "бонусы", "скидка", "скидки",
        "cashback", "кэшбэк", "кешбэк", "постоянный клиент",
        "программа лояльности", "накопительная", "баллы"
    ],
    priority=7,
    facts="""
Программа лояльности Wipon Cashback:

• Начисление бонусов за каждую покупку (настраиваемый %)
• Клиенты оплачивают бонусами до 100% покупки
• Автоматическая идентификация по номеру телефона
• Статистика по постоянным клиентам
• SMS-уведомления о бонусах (опционально)

Результаты внедрения:
- Рост повторных покупок на 25-40%
- Увеличение среднего чека на 15%

Входит в тарифы Standard и Pro.
    """
),
```

**Шаг 3:** Тестируем:
```bash
python scripts/test_knowledge_base.py
# Вводим: "программа лояльности"
# Проверяем что находит нужную секцию
```
